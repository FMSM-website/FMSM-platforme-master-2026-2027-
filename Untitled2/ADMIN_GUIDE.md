# دليل استخدام لوحة التحكم الإدارية

## 🔐 تسجيل الدخول

### بيانات الدخول الافتراضية:
```
اسم المستخدم: admin
كلمة المرور: admin2026
```

⚠️ **مهم جداً:** يُنصح بشدة بتغيير بيانات الدخول!

---

## 🔑 تغيير بيانات تسجيل الدخول

### الخطوة 1: افتح ملف App.tsx
```
src/app/App.tsx
```

### الخطوة 2: ابحث عن هذا القسم (السطر 17 تقريباً):
```typescript
const ADMIN_CREDENTIALS = {
  username: 'admin',
  password: 'admin2026'
};
```

### الخطوة 3: غيّر اسم المستخدم وكلمة المرور:
```typescript
const ADMIN_CREDENTIALS = {
  username: 'your-new-username',    // ← ضع اسم المستخدم الجديد
  password: 'your-strong-password'  // ← ضع كلمة مرور قوية
};
```

### نصائح لكلمة مرور قوية:
- على الأقل 12 حرف
- تحتوي على أحرف كبيرة وصغيرة
- تحتوي على أرقام ورموز
- مثال: `MathSci@2026!WarGla`

---

## 📊 كيفية الوصول للوحة التحكم

1. افتح الموقع
2. اضغط على زر **"الإدارة"** في أعلى اليسار
3. أدخل اسم المستخدم وكلمة المرور
4. اضغط **"تسجيل الدخول"**

---

## 🎛️ مميزات لوحة التحكم

### 1️⃣ تعديل إيميلات الأقسام

**كيفية التعديل:**
1. اضغط على الإيميل الحالي
2. سيتحول إلى حقل نص قابل للتعديل
3. عدّل الإيميل
4. اضغط خارج الحقل لحفظ التغيير
5. اضغط زر **"حفظ التغييرات"** في الأعلى

**مثال:**
- القديم: `master.physics@univ-ouargla.dz`
- الجديد: `deptphy@gmail.com`

---

### 2️⃣ تعديل التخصصات

#### تعديل اسم التخصص:
1. اضغط على اسم التخصص
2. عدّل الاسم في حقل النص
3. اضغط خارج الحقل للحفظ

#### تعديل عدد المقاعد:
1. غيّر الرقم في حقل "المقاعد المتاحة"
2. التغيير يُحفظ تلقائياً

#### تعديل عدد المتقدمين:
1. غيّر الرقم في حقل "عدد المتقدمين"
2. التغيير يُحفظ تلقائياً

---

### 3️⃣ إضافة تخصص جديد

1. اضغط زر **"إضافة تخصص"** في القسم المطلوب
2. سيُضاف تخصص باسم "تخصص جديد"
3. اضغط على الاسم لتعديله
4. أدخل عدد المقاعد وعدد المتقدمين
5. اضغط **"حفظ التغييرات"**

---

### 4️⃣ حذف تخصص

1. اضغط على أيقونة 🗑️ (سلة المهملات) بجانب التخصص
2. ستظهر رسالة تأكيد
3. اضغط **"موافق"** للتأكيد
4. اضغط **"حفظ التغييرات"**

---

### 5️⃣ تعديل التخصص

1. اضغط على أيقونة ✏️ (قلم التحرير)
2. سيفتح وضع التعديل
3. عدّل البيانات المطلوبة
4. اضغط خارج الحقل للحفظ

---

## 💾 حفظ التغييرات

### مهم جداً! 
**التغييرات التي تجريها في لوحة التحكم مؤقتة ولن تُحفظ بشكل دائم إلا إذا:**

#### الطريقة الحالية (يدوية):
1. بعد إجراء التعديلات، اضغط **"حفظ التغييرات"**
2. افتح Console المتصفح (اضغط F12)
3. ابحث عن رسالة "الكود المحدث"
4. انسخ الكود المعروض
5. افتح ملف `src/app/components/DepartmentSelection.tsx`
6. استبدل القسم الخاص بـ `departments` بالكود الجديد

#### قريباً (تلقائية):
سنضيف خاصية حفظ تلقائي في قاعدة بيانات.

## ⚠️ تحذيرات مهمة

1. **لا تشارك بيانات الدخول** مع أي شخص
2. **غيّر كلمة المرور بانتظام** (كل 3 أشهر)
3. **اضغط حفظ** بعد كل تعديل
4. **تحقق من التغييرات** قبل مغادرة اللوحة
5. **لا تحذف** التخصصات التي لديها متقدمين

---

## 🛠️ حل المشاكل

### المشكلة: نسيت كلمة المرور
**الحل:** 
افتح ملف `src/app/App.tsx` واستبدل كلمة المرور في `ADMIN_CREDENTIALS`

### المشكلة: التغييرات لا تُحفظ
**الحل:**
تأكد من الضغط على زر "حفظ التغييرات" ثم اتبع خطوات النسخ اليدوي

### المشكلة: لا يمكن الوصول للوحة التحكم
**الحل:**
تأكد من أنك ضغطت زر "الإدارة" وليس زر آخر
