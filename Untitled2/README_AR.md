# 🎓 منصة الترشح للماستر - جامعة ورقلة

## ✨ المنصة جاهزة بالكامل!

منصة احترافية لإدارة طلبات الترشح للماستر في **كلية الرياضيات وعلوم المادة** بجامعة قاصدي مرباح ورقلة.

---

## 🎯 المميزات الرئيسية

### للطلاب:
✅ **اختيار القسم** - ثلاثة أقسام (فيزياء، كيمياء، رياضيات)
✅ **فئات متعددة** - 5 فئات مختلفة للمترشحين
✅ **نماذج ذكية** - نموذج مخصص لكل فئة
✅ **تأكيد فوري** - رسالة تأكيد بعد إرسال الطلب
✅ **واجهة عربية** - بالكامل من اليمين لليسار

### لرؤساء الأقسام:
✅ **دخول منفصل** - كل رئيس قسم له حسابه الخاص
✅ **عزل الأقسام** - كل رئيس يرى طلبات قسمه فقط
✅ **إحصائيات مباشرة** - عدد الطلبات، قيد المراجعة، المقبولة، المرفوضة
✅ **تنزيل Excel/CSV** - تصدير جميع الطلبات
✅ **فلترة** - حسب الحالة (الكل، قيد المراجعة، إلخ)

## 👥 فئات المترشحين

1. **طلبة ورقلة الجدد** (فئة 80%)
   - نموذج مبسط
   - ترتيب الرغبات
   - لا حاجة للوثائق

2. **طلبة ورقلة السابقين**
   - بيانات كاملة
   - كشوف نقاط السداسيات
   - رفع وثائق إجباري

3. **خارج الجامعة (جدد)**
   - خريجو الجامعات الأخرى حديثاً

4. **خارج الجامعة (سابقين)**
   - خريجو السنوات الماضية

5. **النظام الكلاسيكي**
   - جميع الجامعات - النظام القديم
   - معدلات سنوية بدلاً من سداسية

## 🚀 كيفية الاستخدام

### للطلاب:

1. **افتح الموقع**
2. **اختر القسم** (فيزياء، كيمياء، أو رياضيات)
3. **اختر فئتك** (طلبة ورقلة الجدد، خارجي، إلخ)
4. **املأ النموذج** بجميع البيانات المطلوبة
5. **ارفع الوثائق** (PDF فقط) إن كانت مطلوبة
6. **اضغط "إرسال طلب الترشح"**
7. **انتظر رسالة التأكيد** "تم استلام طلبك بنجاح!"

### لرؤساء الأقسام:

1. **اضغط زر "الإدارة"** (أعلى اليسار)
2. **سجل دخول** بحسابك
3. **شاهد الإحصائيات** في الأعلى
4. **فلتر الطلبات** حسب الحالة
5. **نزّل Excel/CSV** للطلبات
6. **راجع الطلبات** يومياً

## 💾 تنزيل الطلبات

### ملف Excel (.xlsx)
- قابل للفتح في Microsoft Excel
- يحتوي على جميع الحقول
- منسق بشكل احترافي

### ملف CSV (.csv)
- قابل للفتح في Google Sheets
- أخف من Excel
- سهل المعالجة

**اسم الملف:** `اسم_القسم_التاريخ.xlsx`

**مثال:** `قسم_الكيمياء_2026-05-13.xlsx`

## 🗂️ هيكل البيانات

### Firebase Realtime Database:
```
/applications/
  ├── physics/          ← طلبات قسم الفيزياء
  │   ├── -ABC123
  │   ├── -DEF456
  │   └── ...
  ├── chemistry/        ← طلبات قسم الكيمياء
  │   ├── -GHI789
  │   └── ...
  └── mathematics/      ← طلبات الرياضيات
      ├── -JKL012
      └── ...
```

كل طلب يحتوي على:
- معلومات القسم والفئة
- بيانات المترشح الكاملة
- التخصص المرغوب / ترتيب الرغبات
- معلومات الاتصال
- تاريخ ووقت الإرسال
- الحالة (pending, accepted, rejected)

## 🐛 حل المشاكل

### المشكلة: طالب لا يرى رسالة التأكيد
**الحل:** تحقق من console المتصفح (F12) للأخطاء

### المشكلة: رئيس قسم لا يرى الطلبات
**الحل:** 
1. تحقق من Firebase Database
2. تأكد أن الطلبات موجودة تحت `/applications/{departmentId}/`
3. اضغط زر "تحديث"

### المشكلة: ملف Excel فارغ
**الحل:** تأكد من وجود طلبات معروضة في الجدول أولاً
